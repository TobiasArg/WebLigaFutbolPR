# WebLiga Futbol - Base Landing Page

Base inicial para una landing web estatica, lista para personalizar y escalar.

## Estructura

- `index.html`: estructura principal de la landing.
- `styles/main.css`: estilos globales y responsive.
- `scripts/main.js`: interacciones basicas (menu movil y ano en footer).
- `assets/`: recursos estaticos (imagenes, iconos, etc.).

## Ejecutar local

Sin dependencias de Node. Puedes usar un servidor estatico:

```bash
python3 -m http.server 5500
```

Luego abre `http://localhost:5500`.

## Siguientes pasos sugeridos

1. Reemplazar contenido placeholder por tu marca/producto.
2. Agregar formulario de contacto con backend o servicio externo.
3. Integrar analitica (ej. Plausible, GA4).
4. Preparar deploy (Netlify, Vercel, GitHub Pages).
